# C64 Font

TrueType and OpenType fonts containing the Commodore 64 character set.

## Files

- `c64.ttf` - TrueType format
- `c64.otf` - OpenType format (CFF outlines)

## Unicode Coverage

- Basic Latin (A-Z, 0-9, punctuation)
- PETSCII graphics mapped to **Symbols for Legacy Computing** (U+1FB00-U+1FBFF)
- Box drawing characters (U+2500 range)
- Block elements (U+2580 range)
- Card suits, arrows, pi
- Reverse video characters in Private Use Area (U+E000-U+E0FF)
